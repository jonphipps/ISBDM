# has date of publication

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
Record a date that appears in a statement of copyright as a date of publication, if it is considered to be useful for users of the metadata.
</div>

#### Stipulations

<div className="stip">
  Use the element super-type to record a less specific relationship.
</div>
