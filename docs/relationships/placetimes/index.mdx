# Related place and time-span entities

<div className="guid">
Place is an entity that is defined in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink> as "A given extent of space".
</div>

<div className="guid">
Time-span is an entity that is defined in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink> as "A temporal extent having a beginning, an end and a duration".
</div>
