# has entity associated with manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
This element relates a manifestation to an entity at the broadest level of granularity. The related entity may be defined in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink> or in any other entity-relationship model.
</div>

#### Stipulations

<div className="stip">
  Use an appropriate element sub-type to record a more specific relationship between a manifestation and an entity.
</div>
