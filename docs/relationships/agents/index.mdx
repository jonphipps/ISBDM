# Related agent entities

<div className="guid">
Agent is an entity that is defined in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink> as "An entity capable of deliberate actions, of being granted rights, and of being held accountable for its actions".
</div>

<div className="guid">
Collective Agent is an entity that is defined in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink> as "A gathering or organisation of persons bearing a particular name and capable of acting as a unit".
</div>

<div className="guid">
Person is an entity that is defined in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink> as "An individual human being".
</div>

<div className="guid">
Collective Agent and Person are entity sub-types of Agent. A person is an agent; a collective agent is an agent.
</div>

<div className="guid">
A collective agent comprises more than one person.
</div>
