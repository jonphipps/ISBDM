# has person associated with manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
A person may have a form of name that is usually associated with a collective agent. The name may be qualified in an authorized access point to indicate that it is assigned to a person.
</div>

#### Element values

<div className="guid">
The values of this element may be indexed for uncontrolled or controlled keywords, or used to construct browsing lists.
</div>

##### String encoding schemes

<div className="guid">
A string encoding scheme for an authorized access point of a person should support several levels of identification and collation:
  1. A name of the person grouped by family
  2. The dates of birth and death of the person
  3. Specific attributes of the person
</div>

<div className="guid">
<InLink href="/docs/ves/ISBDMSESPer">Person SES</InLink> is a string encoding scheme suggested by ISBD for Manifestation for an authorized access point of a person.
</div>
