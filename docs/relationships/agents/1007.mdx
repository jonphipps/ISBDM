# has collective agent associated with manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
A collective agent may have a form of name that is usually associated with a person. The name may be qualified in an authorized access point to indicate that it is assigned to a collective agent.
</div>

<details>
  <summary>Examples</summary>
  
    | Property | Value |
    |:---------|:------|
    | has collective agent associated with manifestation | "Jethro Tull (Musical group)" |
</details>

#### Element values

<div className="guid">
The values of this element may be indexed for uncontrolled or controlled keywords, or used to construct browsing lists.
</div>

##### String encoding schemes

<div className="guid">
A string encoding scheme for an authorized access point of a collective agent should support several levels of identification and collation:
  1. A name of the collective agent
  2. A category of collective agent
  3. Specific attributes of the collective agent
</div>

<div className="guid">
<InLink href="/docs/ves/ISBDMSESCol">Collective Agent SES</InLink> is a string encoding scheme suggested by ISBD for Manifestation for an authorized access point of a collective agent.
</div>
