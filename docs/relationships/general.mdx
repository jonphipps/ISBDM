# General stipulations for relationship elements

<div className="stip">
  Record a relationship element if it is considered to be useful for users of the metadata, unless indicated otherwise.
</div>

<div className="stip">
  Record a value for an entity that is named or otherwise identified in a statement that appears in the manifestation.
</div>

<div className="stip">
  Record a value for an entity that is associated with the manifestation in external sources of information.
</div>

<div className="stip">
  Record an entity that is probably associated with the manifestation. Record a <InLink href="/docs/notes/1265">has note on entity associated with manifestation</InLink> element that indicates the uncertainty.
</div>

<div className="stip">
  Record an entity that is associated with the manifestation. Do not record an entity that is known to be falsely associated with the manifestation.
</div>

<div className="stip">
  Record an entity that is real and is known to exist or have existed. Do not record an entity that is known to be fictitious.
</div>

<div className="stip">
  Record a <InLink href="/docs/notes/1265">has note on entity associated with manifestation</InLink> element that gives information that is not recorded in a relationship element, if it is considered to be useful for users of the metadata.
</div>

<div className="stip">
  If more than one entity is associated with the same relationship of the manifestation, record a separate occurrence of the element for each distinct value that is considered to be useful for users of the metadata. Use the following order of preference:
  
  1. The prominence of names or titles of entities that appear in statements in the manifestation.
  2. The order of names or titles of entities that appear in a statement in the manifestation.
  3. The predominance of entities that are given in external sources of information.
</div>

<div className="stip">
  For a manifestation that embodies the expression of a diachronic work, and if there is a change in the information that is given in statements that describe the manifestation and appear in manifestations that embody expressions of issues or iterations of the work, record an additional occurrence of the element for the entity that has changed, if it is considered to be useful for users of the metadata.
</div>

<div className="stip">
  For a manifestation that embodies the expression of a diachronic work, record a <InLink href="/docs/notes/1265">has note on entity associated with manifestation</InLink> element that gives information about a change of entity that is related to the manifestations that embody expressions of issues or iterations of the work, including identification of the issue or iteration in which the change first appears, if it is considered to be useful for users of the metadata.
</div>

<div className="stip">
  For a manifestation that embodies the expression of a diachronic work, record a <InLink href="/docs/notes/1265">has note on entity associated with manifestation</InLink> element that gives general information about multiple changes over time of entities that are related to the manifestations that embody expressions of issues or iterations of the work,including identification of the issues or iterations in which the changes occur, if it is considered to be useful for users of the metadata.
</div>
