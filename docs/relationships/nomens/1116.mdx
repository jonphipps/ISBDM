# has chronological designation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
A manifestation may be associated with more than one chronological designation in different languages or scripts, calendars, or sequences.
</div>

<SeeAlso><InLink href="/docs/relationships/1235">has date of creation of manifestation</InLink></SeeAlso>

<SeeAlso><InLink href="/docs/relationships/1117">has numeric designation</InLink></SeeAlso>

#### String encoding schemes

<div className="guid">
For a manifestation that embodies the expression of a diachronic work, a string encoding scheme should combine values for manifestations that embody expressions of issues or iterations of the diachronic work.
</div>

<div className="guid">
<InLink href="/docs/ves/ISBDMSES1116">Chronological designation SES</InLink> is a string encoding scheme suggested by ISBD for Manifestation for a value of this element.
</div>

#### Stipulations

<div className="stip">
  Use an element super-type to record a less specific relationship.
</div>

<div className="stip">
  Apply the stipulations for the element super-type unless indicated otherwise.
</div>

<div className="stip">
  Record a value that is based on a designation in a statement of issue or iteration that appears in the manifestation or an external source of information.
</div>

<div className="stip">
  For manifestation that embodies an expression of an issue or iteration of a diachronic work, record a value that is a chronological designation of only the issue or iteration.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
  |:---------|:------|
  | has chronological designation | "August 2023" |
  
  *[Full example: <InLink href="/docs/fullex/fx032">Mojo 357 (2023; H Bauer Publishing Ltd.; volume + audio disc)</InLink>.]*
  </details>
</div>

<div className="stip">
  For a manifestation that embodies the expression of a diachronic work, record a value for the manifestation that embodies an expression of the first issue or iteration of the work in a chronological sequence, followed by punctuation to indicate an open sequence.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
  |:---------|:------|
  | has chronological designation | "November 1993-" |
  
  *[Full example: <InLink href="/docs/fullex/fx031">Mojo (1993-; H Bauer Publishing Ltd.; volume + audio disc)</InLink>.]*
  <hr />
  
  | Property | Value |
  |:---------|:------|
  | has chronological designation | "2009-" |
  
  *[Full example: <InLink href="/docs/fullex/fx052">Journal für Facility Management (2009-; Technische Universität Wien‏; online resource)</InLink>.]*
  </details>
</div>

<div className="stip">
  For a manifestation that embodies the expression of a diachronic work, and if there is a change in the sequence of chronological designations of issues or iterations, record a value that uses a string encoding scheme to combine:
  
  1. a value for the manifestation that embodies an expression of the first issue or iteration of the work in the old chronological sequence
  2. a value for the manifestation that embodies an expression of the last issue or iteration of the work in the old chronological sequence
  3. a value for the manifestation that embodies an expression of the first issue or iteration of the work in the new chronological sequence
  
  Record a <InLink href="/docs/notes/1273">has note on nomen associated with manifestation</InLink> element that gives information about the change, if it is considered to be useful for users of the metadata.
</div>

<div className="stip">
  For a manifestation that embodies the expression of a diachronic work that is known or assumed to have terminated, record a value that uses a string encoding scheme to combine:
  
  1. a value for the manifestation that embodies an expression of the first issue or iteration of the work in the chronological sequence
  2. a value for the manifestation that embodies an expression of the last issue or iteration of the work in the chronological sequence
</div>
