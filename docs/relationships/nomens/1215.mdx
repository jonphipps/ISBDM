# has fingerprint

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
A fingerprint is an identifier that can be used to detect variant settings of type in otherwise matching manifestations and to identify the reuse of the same setting in ostensibly different manifestations. It can also assist the identification of incomplete items that exemplify the same manifestation.
</div>

<div className="guid">
There is more than one standard method for deriving a fingerprint.
</div>

#### Stipulations

<div className="stip">
