# has nomen associated with manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
This element supports the user tasks to find and identify the manifestation and to explore the context of the manifestation and its relationship to instances of the Nomen entity.
</div>

<SeeAlso><InLink href="/docs/notes/1273">has note on nomen associated with manifestation</InLink></SeeAlso>

#### Element values

<div className="guid">
The values of this element may be indexed for uncontrolled or controlled keywords, or used to construct browsing lists.
</div>

#### Element values

<div className="guid">
The value of a related nomen is a literal string (a nomen string) or an IRI that represents the string.
Record a value that is:
  1. a nomen string
  2. an IRI for the nomen string
##### String encoding schemes
<div className="guid">
An instance of a nomen is a nomen string that may be the result of applying a string encoding scheme to values of attribute and relationship elements that are associated with the entity that is designated by the nomen. A string encoding scheme cannot be applied to an instance of a nomen without generating a separate instance of a nomen.
</div>
<div className="guid">
If a nomen string is the result of applying a string encoding scheme, record a value of a <InLink href="/docs/notes/1273">has note on nomen associated with manifestation</InLink> element that gives information about the string encoding scheme, if it is considered to be useful for users of the metadata.
</div>
</div>

#### Stipulations

<div className="stip">
  Use an appropriate element super-type to record a less specific relationship.
</div>

<div className="stip">
  Use an appropriate element sub-type to record a more specific relationship between a manifestation and a nomen.
</div>

<div className="stip">
  Apply the stipulations for the element super-types unless indicated otherwise.
</div>

<div className="stip">
  For a nomen that is an appellation of an entity other than the manifestation, record a relationship element for the entity or a <InLink href="/docs/notes/1265">has note on entity associated with manifestation</InLink> element or appropriate element sub-type, if it is considered to be useful for users of the metadata.
</div>
