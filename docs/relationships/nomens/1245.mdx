# has variant title of manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
A variant title is selected as an alternative name of the manifestation from all of the titles that appear in the manifestation and external sources of information.
</div>

<div className="guid">
A variant title cannot have the same value as the title proper of the manifestation.
</div>

<div className="stip">
  Use the element super-type to record a less specific relationship.

</div>
