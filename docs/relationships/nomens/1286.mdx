# has uniform resource locator

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
For an online service that is a web page that provides indirect access to a manifestation and does not uniquely identify the manifestation, record the URL in a <InLink href="/docs/notes/1219">has note on category of manifestation</InLink> element.
</div>

<SeeAlso><InLink href="/docs/notes/1219">has note on category of manifestation</InLink></SeeAlso>

#### Element values

<div className="guid">
The value of this element is a string. Record a stringified URL as a value.
</div>
