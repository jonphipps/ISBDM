# has item reproduced by manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
For a manifestation that does not include modifications of an item that exemplifies the manifestation that is copied, use a <InLink href="/docs/relationships/1015">has manifestation reproduced by manifestation</InLink> element.
</div>

<div className="guid">
Use a <InLink href="/docs/relationships/1003">has manifestation associated with manifestation</InLink> element to record the manifestation that is exemplified by the item that is copied.
</div>
