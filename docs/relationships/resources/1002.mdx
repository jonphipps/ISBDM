# has expression associated with manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
This element supports the user tasks to find and identify the manifestation and to explore the context of the manifestation and its relationship to instances of the Expression entity.
</div>

<SeeAlso><InLink href="/docs/notes/1267">has note on expression associated with manifestation</InLink></SeeAlso>

#### Element values

<div className="guid">
The values of this element may be indexed for uncontrolled or controlled keywords, or used to construct browsing lists.
</div>

##### String encoding schemes

<div className="guid">
A string encoding scheme for an authorized access point of an expression should support several levels of identification and collation:
  1. the work that is realized by the expression
  2. a category of content of the expression
  3. specific attributes of the expression
</div>

<div className="guid">
<InLink href="/docs/ves/ISBDMSESExp">Expression SES</InLink> is a string encoding scheme suggested by <span class="thisem">ISBD for Manifestation</span> for an authorized access point of an expression.
</div>
