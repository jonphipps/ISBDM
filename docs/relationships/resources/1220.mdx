# has work embodied in manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
This element is a shortcut for:
has expression embodied in manifestation [Expression] realizes Work
The shortcut does not identify the expression that is embodied in the manifestation. "Realizes" is an Expression element that is the inverse of LRM-R2 in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink>.
</div>

<SeeAlso><InLink href="/docs/assess/p011">Number of expressions</InLink></SeeAlso>

<div className="stip">
  For a manifestation that is an aggregate, and if more than one work is embodied in the manifestation, record one or more occurrences of the element, if it is considered to be useful for users of the metadata. Use the following order of preference to select the values:
  
  - The works that are named most prominently in the typography and layout of a manifestation statement of title and responsibility.
  - The work that is named first in a manifestation statement of title and responsibility.
  - The works that are named in a manifestation statement of title and responsibility in a language and script that is preferred by the cataloguing agency.
  - All works that are embodied in the manifestation.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
