# Related resource entities

<div className="guid">
Expression is an entity that is defined in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink> as "A distinct combination of signs conveying intellectual or artistic content".
</div>

<div className="guid">
Item is an entity that is defined in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink> as "An object or objects carrying signs intended to convey intellectual or artistic content".
</div>

<div className="guid">
Work is an entity that is defined in the <InLink href="https://repository.ifla.org/handle/20.500.14598/40.2">IFLA Library Reference Model</InLink> as "The intellectual or artistic content of a distinct creation".
</div>

<div className="guid">
A related manifestation is a manifestation that is distinct from the manifestation.
</div>
