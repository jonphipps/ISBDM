# has work associated with manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
This element supports the user tasks to find and identify the manifestation and to explore the context of the manifestation and its relationship to instances of the Work entity.
</div>

<SeeAlso><InLink href="/docs/notes/1270">has note on work associated with manifestation</InLink></SeeAlso>

#### Element values

<div className="guid">
The values of this element may be indexed for uncontrolled or controlled keywords, or used to construct browsing lists.
</div>

##### String encoding schemes

<div className="guid">
A string encoding scheme for an authorized access point of a work should support several levels of identification and collation:
  1. an agent who is responsible for the creation of the work
  2. a title that is assigned by an agent who is responsible for the creation of the work or a cataloguing agency
  3. specific attributes of the work
</div>

<div className="guid">
<InLink href="/docs/ves/ISBDMSESWor">Work SES</InLink> is a string encoding scheme suggested by <span class="thisem">ISBD for Manifestation</span> for an authorized access point of a work.
</div>
