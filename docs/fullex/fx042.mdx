# Catalogue of shipwrecked books (2018; William Collins; volume; case binding)

This example describes a printed volume that embodies a work of non-fiction augmented
              with maps and portraits.

The description uses broad elements for an application that does not require finer
              levels of granularity.

The manifestation is published in a single unit.

The manifestation is an augmented expression aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a relational database application.
