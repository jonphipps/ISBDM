# Bridget Riley (2019; National Galleries of Scotland; sheet)

This example describes a printed sheet that embodies a work that is a textual and
              illustrative guide to an art exhibition.

The content is laid out in 4 columns on each page and the sheet is folded to display
              one column at a time.

The manifestation is published in a single unit.

The manifestation is a collected expressions aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a relational database application.
