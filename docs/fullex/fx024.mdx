# كليلة ودمنة (1973؛ الشركة الوطنية للنشر والتوزيع؛ مجلد؛
              غلاف مقوى)

This example describes a printed volume that embodies a collection of works of
              fiction in Arabic, with multiple creators associated with manifestation including
              editors and contributors and two publishers.

The manifestation is published in a single unit.

The manifestation is a collected expressions aggregate.

The manifestation embodies static works.

The language of cataloguing is Arabic. The script of cataloguing is Arabic.

The description is intended for a relational database application.
