# Biblioteka 42 (2000-; Naklada Jesenski i Turk; svezak)

This example describes an incomplete set of printed volumes that embodies a publisher
              series.

The focus of the description is the manifestation of the series, related to the
              manifestations of its issues as sub-units. It is related to a description of the
              issue, inversely related to the series as a super-unit, via authorized access
              points.

The manifestation is published in multiple sub-units.

The manifestation is a collected expressions aggregate.

The manifestation embodies a diachronic work.

The language of cataloguing is Croatian.

The description is intended for a relational database application.
