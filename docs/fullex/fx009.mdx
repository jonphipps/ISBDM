# Lyceum Theatre, London, England, 5/26/72 (2022; Rhino Entertainment LLC; audio
              disc)

This example describes a set of audio discs that embody a work that is a recording of
              a live music concert.

There is no accompanying booklet. All manifestation statements are taken from the CD
              tray holder and disc labels.

The manifestation is published in multiple sub-units.

The manifestation is a collected expressions aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a relational database application.
