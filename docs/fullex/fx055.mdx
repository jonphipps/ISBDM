# Quantum world (2016; Reed Business Information Ltd; volume)

This example describes a printed volume that embodies an issue of a serial. The issue
              reproduces a set of articles that are taken from another diachronic work and
              updated.

The manifestation is also published online.

The manifestation is published in a single unit.

The manifestation is a collected expressions aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for relational database application.
