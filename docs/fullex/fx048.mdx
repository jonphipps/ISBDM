# Loch Morar & Mallaig (2015; Ordnance Survey Ltd; sheet)

This example describes a sheet that embodies a work that is a map.

The map is printed on both sides of the sheet and the sheet is folded.

The manifestation is sub-unit of a series that is not described.

The manifestation is published in a single unit.

The manifestation is not an aggregate.

The manifestation embodies a static work.

The language of cataloguing is English.

The description is intended for a relational database application.
