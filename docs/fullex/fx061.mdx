# National parks (2015; Ziga Media, LLC; flipchart)

This example describes a flipchart that embodies a work that is a calendar. Each leaf
              embodies a reproduction of a poster, and the calendar is augmented with historical
              information about the posters.

The manifestation is a collected expressions and an augmented expression
              aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a relational database application.
