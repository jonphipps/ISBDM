# Who I am (2012; HarperCollinsPublishers; volume; case binding)

This example describes a printed volume that embodies a work of non-fiction that is
              augmented with photographs.

The description uses broad elements for an application that does not require finer
              levels of granularity. The description uses corresponding RDA IRIs for attribute
              values taken from ISBDM value vocabularies that are not yet registered as a
              namespace.

The manifestation is published in a single unit.

The manifestation is an augmented expression aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a linked open data application.
