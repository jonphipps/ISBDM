# Съкровищница (2018; Народна библиотека “Св. св. Кирил и Методий”; том +
              компютърен диск)

This example describes a printed volume that embodies a work of non-fiction augmented
              with facsimile reproduction of manuscripts, old printed books and documents, archival
              materials and photographs with an introductory text. The same content is embodied in a
              printed volume sub-unit in PDF and in a DVD sub-unit. Text content is in Bulgarian in
              Bulgarian Cyrillic script, with parallel text in English.

The manifestation is published in multiple sub-units.

The manifestation is a parallel expressions and an augmented expression
              aggregate.

The manifestation embodies static works.

The language of cataloguing is Bulgarian. The script of cataloguing is Bulgarian
              Cyrillic.

The description is intended for a relational database application.
