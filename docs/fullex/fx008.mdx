# Evangelistarium Marci Maruli Spalatensis (201-; Nacionalna i sveučilišna
              knjižnica u Zagrebu; online jedinica građe)

This example describes an online resource that is a digitised reproduction of an
              older printed volume that embodies a work of non-fiction.

The manifestation is a sub-unit of a series of digitised reproductions that is not
              described. The manifestation reproduces a specific item, including modifications such
              as annotations and visible wear of binding and pages.

The manifestation is published in a single unit.

The manifestation is not an aggregate.

The manifestation embodies a static work.

The language of cataloguing is Croatian.

The description is intended for a relational database application.
