# Matisse Bonnard (2017; Städel Museum; volume)

This example describes a printed volume that embodies a work that is a catalogue of
              an exhibition of two painters with reproductions of their paintings and photographs
              augmented with a chronology, list of exhibited works and selected bibliography.

The manifestation is published in a single unit.

The manifestation is an augmented expression and a collected expressions
              aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a relational database application.
