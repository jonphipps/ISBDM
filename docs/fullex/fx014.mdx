# Was mir behagt, ist nur die muntre Jagd (2021; Carus; volume; full score)

This example describes a printed volume that embodies a work that realizes multiple
              expressions of notated music augmented with critical notes in German.

The prominent manifestation statements are in parallel German and English. The
              manifestation is sub-unit of a series which is not described.

The manifestation is published in a single unit.

The manifestation is a collected expressions aggregate and an augmented expression
              aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a relational database application.
