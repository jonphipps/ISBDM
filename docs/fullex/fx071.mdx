# Pogled na Akademijin vrt (195-; Mujadžić, Omer, 1903-1991; sheet)

This example describes a sheet that embodies a work that is a pastel painting.

No statements appear in the manifestation.

The manifestation is produced in a single unit.

The manifestation is not an aggregate.

The manifestation embodies a static work.

The language of cataloguing is English.

The description is intended for a relational database application.
