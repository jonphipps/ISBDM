# Secret library (2016; Michael O’Mara Books Limited; volume; case binding)

This example describes a printed volume that embodies a work of non-fiction that is
              augmented with an index.

The manifestation is published in a single unit.

The manifestation is an augmented expression aggregate.

The manifestation embodies static works.

The language of cataloguing is English.

The description is intended for a linked open data application.
