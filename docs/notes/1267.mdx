# has note on expression associated with manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
Record a value of a broad kind of expression that is embodied in a manifestation as a value of <InLink href="/docs/attributes/1264">has category of embodied content</InLink>.
</div>

<div className="guid">
For a manifestation that is an aggregate, record a quantitative value for a kind of expression that is embodied in the manifestation as a value of <InLink href="/docs/attributes/1278">has extent of aggregated content</InLink>.
</div>

<SeeAlso><InLink href="/docs/attributes/1264">has category of embodied content</InLink></SeeAlso>

<SeeAlso><InLink href="/docs/relationships/1002">has expression associated with manifestation</InLink></SeeAlso>

<SeeAlso><InLink href="/docs/attributes/1278">has extent of aggregated content</InLink></SeeAlso>

#### Stipulations

<div className="stip">
  Use the element super-type to record a less specific note.
</div>

<div className="stip">
  Apply the stipulations for the element super-type unless indicated otherwise.
</div>

<div className="stip">
