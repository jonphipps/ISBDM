#### Stipulations

<div className="stip">
  Use the element super-type to record a less specific note.
</div>

<div className="stip">
  Apply the stipulations for the element super-type unless indicated otherwise.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
  |:---------|:------|
  | has note on work associated with manifestation | "A selection from Written in their soul: the Stax songwriter demos (7-CD)." |
  
  *[Full example: <InLink href="/docs/fullex/fx034">Stax uncovered! (2023; H Bauer Publishing Ltd; audio disc)</InLink>.]*
  <hr />
  
  | Property | Value |
  |:---------|:------|
  | has note on work associated with manifestation | "Me të dhëna historike, topografike, demografike e statistikore sipas besimit për vendbanime shqiptare, përfshirë edhe Kosovën." |
  
  *[Full example: <InLink href="/docs/fullex/fx074">Corso delli fiumi Drino, e Boiana nella Dalmatia (1688; fletë)</InLink>.]*
  </details>
</div>

<div className="stip">
  For a manifestation that is an older printed sheet or volume, record a value that gives a standard citation to a work that includes a structured description of the manifestation.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
  |:---------|:------|
  | has note on work associated with manifestation | "Bibliographic reference: Schramm, A., Bilderschmuck, v. 4, p. 10, 50, and ill." |
  </details>
</div>

<div className="stip">
  For a manifestation that embodies the expression of a diachronic work, record a value that gives information about the following characteristics of the diachronic work:
  
  - Frequency of issues or iterations of the diachronic work.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
  |:---------|:------|
  | has note on work associated with manifestation | "Frequency is irregular." |
  
  <hr />
  
  | Property | Value |
  |:---------|:------|
  | has note on work associated with manifestation | "Issued monthly." |
  
  *[Full example: <InLink href="/docs/fullex/fx031">Mojo (1993-; H Bauer Publishing Ltd.; volume + audio disc)</InLink>.]*
  <hr />
  
  | Property | Value |
  |:---------|:------|
  | has note on work associated with manifestation | "Frequency unknown." |
  </details>
</div>
