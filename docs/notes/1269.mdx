#### Stipulations

<div className="stip">
  Use the element super-type to record a less specific note.
</div>

<div className="stip">
  Apply the stipulations for the element super-type unless indicated otherwise.
</div>

<div className="stip">
  Record a value that gives information about other manifestations that embody the same content in a different category of carrier.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
  |:---------|:------|
  | has note on manifestation associated with manifestation | "Also published online and on microfilm." |
  
  <hr />
  
  | Property | Value |
  |:---------|:------|
  | has note on manifestation associated with manifestation | "Also issued as glass slides." |
  </details>
</div>

<div className="stip">
  Record a <InLink href="/docs/relationships/1018">has alternate manifestation</InLink> element that relates the manifestation to another manifestation that is published or produced at the same time and that embodies the same content.
</div>

<div className="stip">
  Record a value that gives information about the bibliographic history of the manifestation, including previous manifestations that are related to the manifestation.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
  |:---------|:------|
  | has note on manifestation associated with manifestation | "Originally published: London : J. Tonson, 1747." |
  
  <hr />
  
  | Property | Value |
  |:---------|:------|
  | has note on manifestation associated with manifestation | "Previously published as: The players of Null-A. London : Dodson, 1970; originally published as: The pawns of Null-A. New York : Street and Smith, 1948." |
  </details>
</div>

<div className="stip">
  For a manifestation that reproduces another manifestation, record a value that gives information about the manifestation that is reproduced.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
  |:---------|:------|
  | has note on manifestation associated with manifestation | "Facsimile reproduction. Originally published: London : R. Walton, 1654." |
  
  <hr />
  
  | Property | Value |
  |:---------|:------|
  | has note on manifestation associated with manifestation | "Reproduction of a drawing created in 1522." |
  </details>
</div>

<div className="stip">
  For a manifestation that is published or produced in more than one unit, record a value that gives information about one or more sub-units or super-units.
  
  <details>
  <summary>Examples</summary>
  
  | Property | Value |
  |:---------|:------|

  </details>

<div className="stip">

  <details>
  <summary>Examples</summary>
  
  | Property | Value |

  </details>
