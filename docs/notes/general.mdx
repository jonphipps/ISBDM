# General stipulations for note elements

<div className="stip">
  Record a value if it is considered to be useful for users of the metadata.
</div>

<div className="stip">
  Record a value that is well-formed in the orthography of the language and script of the cataloguing agency.
</div>

<div className="stip">
  Record a value that terminates in a full-stop or equivalent punctuation mark.
</div>

<div className="stip">
  Record a value that is unstructured free-text.
</div>

<div className="stip">
  Record a value that follows a standard pattern.
</div>

<div className="stip">
  Record a separate occurrence of the element for each distinct value that is considered to be useful for users of the metadata.
</div>

<div className="stip">
  Record a combination of notes in a single occurrence of the element, if it is considered to be useful for users of the metadata.
</div>

<div className="stip">
  Record multiple occurrences of the element in an order that is considered to be useful for users of the metadata.
</div>
