# Note elements

<div className="guid">
Use a note element to record a string value that is supplied by a cataloguing agency.
</div>

<div className="guid">
The value supports the general principles of "accuracy", "sufficiency and necessity", "significance", and "economy" of <InLink href="https://repository.ifla.org/handle/20.500.14598/80">Statement of international cataloguing principles (ICP)</InLink>.
</div>
