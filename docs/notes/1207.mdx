# has note on manifestation

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
This element supports the user task to identify the manifestation.
</div>

#### Element values

<div className="guid">
The values of this element may be indexed for uncontrolled keywords to support the user task to find the manifestation.
</div>

<div className="guid">
For a manifestation that embodies the expression of a diachronic work, include a value of any of the following elements for the identification in a note of the manifestation that embodies an expression of an issue or iteration of the work:
  - <InLink href="/docs/relationships/1116">has chronological designation</InLink>
  - <InLink href="/docs/relationships/1210">has date of publication</InLink>
  - <InLink href="/docs/relationships/1117">has numeric designation</InLink>
</div>

<div className="guid">
Use a <InLink href="/docs/notes/1268">has note on item associated with manifestation</InLink> element to record information about a modification of an item that exemplifies the manifestation after its publication or production.
</div>

#### Stipulations

<div className="stip">
  Apply the <InLink href="/docs/notes/general">General stipulations for note elements</InLink>.
</div>

<div className="stip">
