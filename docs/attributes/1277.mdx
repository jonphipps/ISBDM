# has extent of embodied content

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<SeeAlso><InLink href="/docs/attributes/1288">has layout of embodied content</InLink></SeeAlso>

#### Element values

<div className="guid">
For an extent of a cartographic image, cartographic tactile image, still image, or tactile image, round a fraction of the unit of length to the nearest whole number.
Record a <InLink href="/docs/notes/1284">has note on physical characteristic</InLink> element that gives a more precise measurement, if it is considered to be useful for users of the metadata.
<details>
  <summary>Examples</summary>
  
    | Property | Value |
    |:---------|:------|
    | has extent of embodied content | "60 cm × 81 cm" |
    | has note on physical characteristic | "Dimensions of still image: 60.33 cm × 81.12 cm" |
    
    *[Full example: <InLink href="/docs/fullex/fx045">Treachery of images (ceci n'est pas une pipe) (1929; Magritte, René, 1898-1967; sheet)</InLink>.]*
</details>
</div>

  <details>
