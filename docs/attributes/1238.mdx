# has terms of availability

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
For a manifestation that embodies the expression of a diachronic work, consider information about the frequency and pattern of issues to be publication metadata and not terms of availability
</div>

<SeeAlso><InLink href="/docs/statements/1034">has manifestation statement of identifier and terms of availability</InLink></SeeAlso>

#### Stipulations

<div className="stip">
  Use the element super-type to record a less specific attribute.
</div>

<div className="stip">
  Apply the stipulations for the element super-type unless indicated otherwise.
