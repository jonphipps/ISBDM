</div>

#### Element values

<div className="guid">
The values of this element may be indexed for controlled keywords.
</div>

<div className="guid">
The values of this element may be parsed to extract carrier and content metrics.

#### Element values

<div className="guid">
The value of this element is a quantitative value that measures a single characteristic, or a combination of quantitative and qualitative values that measure multiple characteristics.
Apply a string encoding scheme to construct a combination of values.
</div>

<div className="guid">

</div>
