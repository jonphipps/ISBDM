# Attribute elements

<div className="guid">
Use an attribute element to record a controlled or uncontrolled string value that is supplied by a cataloguing agency.
</div>

<div className="guid">
The value supports the general principles of "accuracy", "sufficiency and necessity", "significance", "economy", "integration", and "interoperability" of <InLink href="https://repository.ifla.org/handle/20.500.14598/80">Statement of international cataloguing principles (ICP)</InLink>.
</div>
