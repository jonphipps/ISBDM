# has extent of aggregated content

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
Record an uncounted kind of expression that is embodied in the manifestation as a value of <InLink href="/docs/attributes/1264">has category of embodied content</InLink> or as a value of <InLink href="/docs/notes/1267">has note on expression associated with manifestation</InLink>.
</div>

<div className="guid">
For a manifestation that embodies only one expression, consider the extent to be derived as the quantity "1" of a value that is recorded for the <InLink href="/docs/attributes/1264">has category of embodied content</InLink> element.
</div>

<SeeAlso><InLink href="/docs/attributes/1264">has category of embodied content</InLink></SeeAlso>

<SeeAlso><InLink href="/docs/notes/1267">has note on expression associated with manifestation</InLink></SeeAlso>

#### Stipulations

<div className="stip">
  Use the element super-type to record a less specific attribute.
</div>

<div className="stip">
  Apply the stipulations for the element super-type unless indicated otherwise.
</div>

<div className="stip">
  Record a quantitative value that consists of a whole number and a term that appears in the <InLink href="/docs/ves/1264">ISBDM Category of Content value vocabulary</InLink> for the category of content that is counted, if it is considered to be useful for users of the metadata.
  For a finer level of granularity, record a quantitative value that consists of a whole number and a term that appears in the <InLink href="/docs/ves/1278">ISBDM Extent of Aggregated Content value vocabulary</InLink> for the kind of content that is counted, if it is considered to be useful for users of the metadata.
