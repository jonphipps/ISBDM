# has number of sound channels

## Element Reference
<ElementReference frontMatter={frontMatter} />

#### Additional information

<div className="guid">
Record the number of channels that embody the content. Do not include recording tracks that are not used.
</div>

<div className="guid">
Consider a ".1" channel for a subwoofer as a distinct channel.
</div>

<SeeAlso><InLink href="/docs/attributes/1285">has designation of number of sound channels</InLink></SeeAlso>

</div>
