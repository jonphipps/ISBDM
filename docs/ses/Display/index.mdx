# Display of ISBDM metadata

<div className="guid">
**ISBD for Manifestation** does not prescribe a selection, order, layout, or punctuation for displaying the values of elements in a description of a manifestation.
</div>

<div className="guid">
The choice of methods for displaying ISBDM metadata in an information retrieval application is influenced by the intended <InLink href="/docs/intro/i006">Metadata utility and processing</InLink> of ISBDM elements.
</div>

<div className="guid">
The value of a has manifestation statement element or element sub-type is at a broad level of granularity, and it may overlap with the value of another similar element.
</div>

<div className="guid">
Specific element labels may be omitted to simplify the display of a set of values.
</div>

<div className="guid">
The controlled value of an attribute element can be displayed or otherwise output as a non-textual symbol.
</div>

<details>
<summary>Examples</summary>

| Relation | Example |
|----------|----------|
| display of ISBDM metadata | "Display the value "volume" of a has category of manifestation element as ." |

---

| Relation | Example |
|----------|----------|
| display of ISBDM metadata | "Display the value "microform" of a has media type element as ." |

</details>

<div className="guid">
The controlled values of a <InLink href="/docs/attributes/1263">has category of manifestation</InLink> element and element sub-types can be concatenated into a single compound value and displayed with or without a generic "category" label to simplify the display of a set of values.
</div>

<details>
<summary>Examples</summary>

| Relation | Example |
|----------|----------|
| display of ISBDM metadata | ""volume; unmediated; still image; text"" |

*[Full example: <InLink href="/docs/fullex/fx059">Asterix and the gowden heuk (2014; Itchy Coo; volume)</InLink>. The delimiting punctuation is "; " (semicolon-space).]*

---

| Relation | Example |
|----------|----------|
| display of ISBDM metadata | ""Type: audiocassette - spoken word"" |

*[Full example: <InLink href="/docs/fullex/fx018">Harry Potter and the goblet of fire (2000; Listening Library; audiocassette)</InLink>. The delimiting punctuation is " - " (space-hyphen-space). The value of <InLink href="/docs/attributes/1218">has media type</InLink> is omitted for clarity. "Type" is used as a generic display label.]*

</details>

<div className="guid">
The controlled values of a <InLink href="/docs/relationships/1019">has creator agent of manifestation</InLink>, has place of creation of manifestation, and has date of creation of manifestation elements and element sub-types can be concatenated into a single compound value and displayed with or without a generic "publication" label to simplify the display of a set of values.
</div>
