<div className="guid">

This is a suggested string encoding scheme for an authorized access point of an item that is related to the manifestation.

The encoding scheme is applicable to the value of a <InLink href="/relationships/1004">has item associated with manifestation</InLink> element or element sub-type.

</div>

<div className="guid">

The base access point of an item is a value of <InLink href="/relationships/1257">has authorized access point of manifestation</InLink>.

The base access point is followed by a qualifier that is added to the qualifiers of the access point and separated by a semicolon "; ".

The ISBDM string template is:

`{has authorized access point of manifestation}` + insert before ")"("; " + [an identifier or shelfmark of the item])

The qualifier supports a local secondary collocation and sort order of authorized access points of manifestations in a browse index for items.
