<div className="guid">

This is a suggested string encoding scheme for an authorized access point of an agent who is related to the manifestation.

The encoding scheme is applicable to the value of a <InLink href="/relationships/1005">has agent associated with manifestation</InLink> element or element sub-type.

</div>

<div className="guid">

For an agent who is a collective agent, apply the <InLink href="/ses/col">Collective Agent SES</InLink>.

</div>

<div className="guid">

For an agent who is a person, apply the <InLink href="/ses/per">Person SES</InLink>.

</div>

<div className="guid">

For any agent, add a qualifier that indicates that the agent is a collective agent or a person.

</div>
