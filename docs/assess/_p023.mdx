### Manifestation that is a single unit {#p023}

<section className="guid">

For a manifestation that is published or produced in only one unit, record a single description.

</section>

<section className="guid">

Record a <InLink href="/docs/attributes/1022">has category of carrier</InLink> element. This is mandatory.

Describe other physical characteristics of the unit by applying any of the following methods:

- Record a <InLink href="/docs/attributes/1242">has physical characteristic</InLink> element or element sub-type.
  - Record a <InLink href="/docs/attributes/1023">has extent of manifestation</InLink> element or element sub-type.

- Record a <InLink href="/docs/notes/1284">has note on physical characteristic</InLink> element.
- Record a <InLink href="/docs/statements/1025">has manifestation statement</InLink> element or element sub-type that includes information about the unit.

Describe the unit with more than one method, if it is considered to be useful for users of the metadata.
