### Manifestation that embodies an issue of a diachronic work {#p020}

<section className="guid">

Consider the manifestation that embodies an expression of an issue of a diachronic work to be a sub-unit of the incomplete manifestation that embodies the expression of the diachronic work.

</section>

<section className="guid">

Select the methods to use to describe the manifestation of the diachronic work from the following options:

1. Record the manifestation that embodies the diachronic work as a separate manifestation and record a relationship to it with a <InLink href="/docs/relationships/1014">has super-unit</InLink> element.
2. Record a <InLink href="/docs/statements/1025">has manifestation statement</InLink> or element sub-type that includes information about the manifestation that embodies the diachronic work.
3. Record a <InLink href="/docs/notes/1269">has note on manifestation associated with manifestation</InLink> element that includes a description of the manifestation that embodies the diachronic work.
4. Do not describe the manifestation that embodies the diachronic work.

Describe the manifestation of the diachronic work with more than one method, if it is considered to be useful for users of the metadata.
