# Examples of transcription

<div className="guid">
These examples illustrate the application of the <InLink href="/docs/statements/transcription">Rules for transcribing a manifestation statement</InLink> to the value of a <InLink href="/docs/statements/1025">has manifestation statement</InLink> element or element sub-type.
</div>

<div className="guid">

    | Property | Value |
    |:---------|:------|
    | has manifestation statement of publication, production, manufacture, or distribution | "London: Printed for John Murray. 1816" |
    
    *[Full example: <InLink href="/docs/fullex/fx053">Emma (1815; John Murray; volume)</InLink>. The value transcribes the punctuation of the statement as it appears in the manifestation. Whitespace is not replaced by punctuation.]*
</div>

<div className="guid">
  
    | Property | Value |
    |:---------|:------|
    | has manifestation statement of publication, production, manufacture, or distribution | "First published in the United Kingdom in 2021 by Thames & Hudson … London …" |
    
    *[Full example: <InLink href="/docs/fullex/fx063">Spring cannot be cancelled (2022; Thames & Hudson; volume)</InLink>. The value omits the address and postal code that are considered to be unnecessary detail, and replaces them with marks of omission "…".]*

</div>

<div className="guid">

    | Property | Value |
    |:---------|:------|
    | has manifestation statement of title and responsibility | "Corso delli Fiumi Drino e Boiana nella Dalmatia / descritto dal P. Coronelli, cosmografo della Seren. Republica di [...]" |
    
    *[Full example: <InLink href="/docs/fullex/fx077">Corso delli Fiumi Drino e Boiana nella Dalmatia (2007; <br />Bibliothèque nationale de France; online resource; PDF)</InLink>. The value retains the punctuation of the statement. The mark of omission "..." in brackets is transcribed from the statement. It does not indicate an omission in transcription, and it is not encoded as an ellipsis.]*

</div>

<div className="guid">
  
    | Property | Value |
    |:---------|:------|
    | has manifestation statement of title and responsibility | "Matisse Bonnard, Long live painting!" |
    
    *[Full example: <InLink href="/docs/fullex/fx065">Matisse Bonnard (2017; Städel Museum; volume)</InLink>. The value interprets the layout and typography to include a space between the first two words. Another value could interpret the layout to include whitespace, and may replace it with clarifying punctuation.]*

</div>

<div className="guid">
  
    | Property | Value |

    | has manifestation statement of publication, production, manufacture, or distribution | "©2022 Aimee Stwart" |
    
    *[Full example: <InLink href="/docs/fullex/fx068">Library (2022; Corner Piece Puzzles; object)</InLink>. The value includes a misprint; the letter "e" is missing from "Stwart".]*

</div>

<div className="guid">
  
    | Property | Value |

    *[Full example: <InLink href="/docs/fullex/fx074">Corso delli fiumi Drino, e Boiana nella Dalmatia (1688; fletë)</InLink>. The value retains abbreviations and does not expand them. The value replaces whitespace with the punctuation ", " (comma-space) for clarity.]*

</div>
