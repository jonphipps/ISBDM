# General stipulations for manifestation statement elements

<div className="stip">
  Record a value that appears in the manifestation.
  Do not record a value that does not appear in the manifestation.
</div>

<div className="stip">
  If more than one statement is associated with the same aspect of the manifestation, record a separate occurrence of the element for each distinct statement that is considered to be useful for users of the metadata. Use the following order of preference:
  
  1. The prominence of statements that appear in the manifestation.
  2. The fullness of statements within the scope of the element.
  3. The utility of statements for users of the metadata.
  4. The order of statements that appear in the manifestation.
</div>

<div className="stip">
  If a statement is duplicated in different languages or scripts, record a separate occurrence of the element for each distinct statement that is considered to be useful for users of the metadata. Use the following order of preference:
  
  1. The languages and scripts that are preferred by the cataloguing agency.
  2. The order of statements that appear in the manifestation.
</div>

<div className="stip">
  For a manifestation that embodies the expression of a diachronic work, record a value that describes only the manifestation. Omit a statement or part of a statement that describes a manifestation that embodies an expression of an issue of the work.
</div>

<div className="stip">
  For a manifestation that embodies the expression of a diachronic work, and if there is a change in the information that is given in statements that describe the manifestation and appear in manifestations that embody expressions of issues or iterations of the work, record an additional occurrence of the element for the statement that has changed, if it is considered to be useful for users of the metadata.
</div>

<div className="stip">
  For a manifestation that embodies an expression of an issue of a diachronic work, record only a value that describes the manifestation. Record a statement or part of a statement that describes the manifestation of the expression of the diachronic work as a value of a <InLink href="/docs/statements/1031">has manifestation statement of issue or iteration</InLink> element.
</div>
