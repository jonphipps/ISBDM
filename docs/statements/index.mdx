# Statement elements

<div className="guid">
Use a manifestation statement element to record a string value that is transcribed from the manifestation.
</div>

<div className="guid">
The value supports the general principle of "representation" of <InLink href="https://repository.ifla.org/handle/20.500.14598/80">Statement of international cataloguing principles (ICP)</InLink>.
</div>
